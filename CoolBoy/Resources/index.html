Welcome to CoolBoy

The simple C# Webserver.

Before starting...

CoolBoy needs to be run as Administrator, otherwise it won't be able to work.

The setup.bat file can be run to create a firewall exception on port 80. Only run it if you know what you're doing.

Configuration

In the Preferences window, you can select the folder where your HTML files are stored, as well as the default main file (which is index.html by default).

That's all folks

Don't expect too much from CoolBoy, though, as it is mostly a toy.
